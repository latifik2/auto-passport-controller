Этот проект является контроллером Kubernetes, предназначенный для автоматического создания документации (паспортов) сервисов.

Цель данного проекта - снизить трудочасы сотрдуников на ведение документации и попытаться максимально автоматизировать этот процесс.

На данный момент запланирована реализации автоматического документирования для четырех сервисов:
- Airflow
- Metabase
- Supset
- Произвольные пользовательские приложения/микросервисы

Реализованные способы обнаружения приложени:
- из конфига - done
- в кластерах Kubernetes - done
- через consul - not done

Реализованна поддержка:
- Airflow - done
- Metabase - not done
- Superset - not done
- Микросервисы - not done
